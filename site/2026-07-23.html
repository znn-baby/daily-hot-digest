AI 智能体与开发范式

今天 GitHub 上 AI Agent 相关的动静可不少。worldmonitor[1] 用 AI 做实时全球情报监控与地缘政治追踪，已经拿了 66,000+ star，搞国际关系研究的朋友或者对实时情报感兴趣的人可以看看。ai-agent-book[2] 是李博杰写的智能体设计开源书，配套完整代码，想系统入门的话这本不错。diegosouzapw/OmniRoute[9] 这个项目也很有意思，号称一个 Endpoint 就能调用 278+ 个提供商的模型，还内置了自动降级和 Token 压缩，对于需要在不同大模型间无缝切换的开发者来说简直是福音。

系统与底层安全

底层技术这块儿有点硬核。

Lobsters 上有一篇关于 RefluXFS 的帖子 [16]，说是一个 Linux 内核本地提权漏洞，搞安全的朋友必看。另外 LG 宣布要禁止在智能电视应用中使用住宅代理 [11]，这事挺有意思的，感觉是厂商在收紧对 TV 端数据抓取和隐私保护的控制。还有 PyPI 发布新机制，14 天后拒绝新文件 [10]，这对依赖 PyPI 的 Python 开发者来说是个挺重要的部署流程变更。

数码产品与效率工具

数码圈今天动静挺大，

少数派报道了三星发布多款折叠屏新品和索尼 FX5 电影摄影机 [25]，还有 DJI Pocket 4P 的漫谈 [1]，感觉折叠屏和手持云台这波升级挺猛的。效率工具方面，ruvnet/RuView[3] 利用 commodity WiFi 信号做无视频的生命体征监测，这思路很清奇，不需要摄像头就能感知空间状态。另外 Pip 26.2 更新解决了困扰 16 年的部署 hack 问题 [24]，对于部署 Python 应用的老手来说，这绝对是省心的大更新。
